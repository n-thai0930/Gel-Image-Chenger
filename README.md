# Gel-Image-Chenger

This software extracts only the gel portion from electrophoresis photographs. It supports any image format.
このソフトウェアは電気泳動の写真からゲルの部分だけを切り出してくるソフトウェアです。どんな写真の形式にも対応しています。

[Release](https://github.com/n-thai0930/Gel-Image-Chenger/releases/tag/v1.0)からzipをDLしていただくと、GUI付きのソフトウェアをダウンロードしていただけます
